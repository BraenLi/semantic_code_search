# QENAS - 量子纠缠生态位自适应网络

## 项目概述

QENAS 是一个基于复杂系统科学、量子认知理论、生物进化算法和行为金融学的革命性量化交易策略系统。

## 核心特性

| 维度 | 传统策略 | QENAS |
|------|---------|-------|
| 相关性建模 | 线性相关系数 | 量子纠缠（非线性、非局域） |
| 策略多样性 | 单一或少数策略 | 100+ 生态位分离的策略物种 |
| 适应性 | 定期重训练 | 实时共同进化 |
| 决策机制 | 规则或简单投票 | 涌现同步（Kuramoto 振子） |
| 风险管理 | 静态止损 | 临界相变预警 |

## 项目结构

```
QENAS/
├── qenas_core/            # 核心策略引擎
│   ├── quantum/           # 量子纠缠层
│   ├── ecosystem/         # 生态位子策略群
│   ├── neuromorphic/      # 神经形态处理器
│   ├── emergent/          # 涌现决策引擎
│   └── strategies/        # 策略实现
├── qenas_data/            # 数据服务
│   ├── feeders/           # 数据源 (A 股/美股/港股)
│   ├── storage/           # 数据存储
│   └── processors/        # 数据预处理
├── qenas_backtest/        # 回测引擎
├── qenas_web/             # FastAPI 后端
├── qenas_ui/              # Vue3 前端
├── tests/                 # 单元测试
├── pyproject.toml
├── README.md
└── QENAS_DEVLOG.md        # 开发日志
```
